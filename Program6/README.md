# Program 6: K-Means Clustering


## Introduction
This program demonstrates the implementation of the K-Means clustering algorithm, an unsupervised machine learning technique used to partition data into k distinct clusters.


## Algorithm Overview
1. Initialize k centroids randomly.
2. Assign each data point to the nearest centroid.
3. Update the centroids by calculating the mean of all points assigned to each cluster.
4. Repeat steps 2 and 3 until the centroids no longer change.


## Files Included

- **p6.ipynb**: Main notebook containing the K-Means implementation and detailed comments.

- **6Kmeans.ipynb**: Secondary notebook with short/alternative code for K-Means.
