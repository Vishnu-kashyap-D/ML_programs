# Machine Learning Lab Repository


## Welcome
Welcome to my 5th Semester ML Lab practicals repository.


This repository contains Jupyter notebooks for various Machine Learning algorithms.


## Programs Included

- **Program 1**: Introduction to Data Processing and Graphing

- **Program 2**: Linear Regression Implementation

- **Program 3**: Logistic Regression Implementation

- **Program 4**: K-Nearest Neighbors (KNN)

- **Program 5**: Decision Trees (DT)

- **Program 6**: K-Means Clustering

- **Program 7**: Support Vector Machines (SVM)

- **Program 8**: Principal Component Analysis (PCA)


## Repository Structure

Each ML algorithm has its own dedicated directory (e.g., Program1, Program2, etc.).

Most directories contain both the main detailed notebook and a shorter/alternative notebook version.


## Prerequisites and Dependencies

To run these notebooks locally, you will need the following installed:

- **Python 3.x**

- **Jupyter Notebook** or JupyterLab

- **NumPy** & **Pandas** for efficient data manipulation.

- **Matplotlib** & **Seaborn** for visualizations and plotting.

- **Scikit-Learn** for standard machine learning models.
